Process a JSONL file by applying function f to each entry and handling retries.

Parameters:

  • file_path – Path to the JSONL file.

  • f – Function to generate new JSON objects.

  • arg_generators – Generators that produce arguments for the function f.

  • interval – Time interval (in seconds) between saving results.

  • retry – Whether to retry based on function should_retry.

  • should_retry – Function to determine if the result from f should be retried.
